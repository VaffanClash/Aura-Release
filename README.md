## Support
The app supports:
- Steam games
- Ubisoft games
- Epic games
- files

Is has been fully tested for
- Steam games
- Ubisoft games

## Next steps

## How to use
- Simply Drag&Drop the game's shortcut into the window
- You can move the app around by dragging it by the very top part of the window.
